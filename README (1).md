# Muhammad Hamza Rashid

A collection of small AI/Claude projects — custom skills, connector integrations, and exploratory conversations with Claude.

## Projects

### 1. Project Docs Generator (Custom Claude Skill)
A custom Claude skill (`project-docs-generator`) built to automatically generate a professional 3-file documentation package (README, prompts log, and tools log) for AI/skill projects — so they can be uploaded straight to GitHub without manually writing a README each time.

### 2. Gmail Connector — Inbox Search & Summarization
Documents connecting Claude to a Gmail account through Anthropic's connector system, enabling Claude to search the inbox and read/summarize real email threads on request.

### 3. Gmail Connector — Live Data Verification
Documents connecting Claude to a Gmail account through Anthropic's connector system, and verifying that the connection returns real, current inbox data by fetching the last 2 AWS-related emails.

### 4. Shared Doc Generator Skill
Shared the "doc generator skill" (Project 1) with another person, who ran it and got the same results, confirming the skill works consistently across users.

### 5. MCP Builder Skill — Plain-English Walkthrough
Documents a conversation asking Claude to explain the `mcp-builder` skill in plain English and flag anything sensitive about it — including external contact, credential handling, and unexpected data flow.

## AI Tools / Apps Used
- **Claude (Anthropic)** — core assistant used to design, build, and test all projects
- **Claude Skills** — custom skill authoring/execution (Project 1)
- **Gmail Connector (Anthropic MCP integration)** — inbox access and email search/summarization (Projects 2 & 3)
- **MCP Builder** — reviewed/explained as part of Project 5
